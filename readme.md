An example solution of learning ASP .NET MVC  
Need SQL SERVER (EXPRESS) installed to run, config Database Connection string in Web.config  
Use entity framework to deal with db, restore nuget packages before compile project  
Router is configured in RouteConfig.cs  
BundleConfig registers  resources such as js or css files as a bundle  
Migrations defines how to do database migration and seed method  
Scripts contains library or custom js files
